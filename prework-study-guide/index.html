HTML

  • Enter your HTML notes here
    • The 'head' element contains information about the webpage.
      • The head element also contains the metadata for the web application.
      • Metadata is data about the webpage that the browser uses to effectively serve our webpage to users. Metadata is typically not seen by users, but it contains important information about accessibility, search engines and performance.
        • Metadata such as character set, description, keywords, author and viewport are defined by the 'meta' element.
    • The 'body' element represents the visible content shown to the user.
    • The 'tittle' element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO).
    • The 'style' element defines the CSS styles associated with a website.
    • The 'base' element defines the base URL for a webpage.
    • The 'link' element connets an external resource to the HTML document.
    • The 'script' element embeds client-side scripts in the HTML document.
    • The 'viewport' element sets the default view for our webpage.

CSS

  • Enter your CSS notes here
  • A 'margin' indicates how much space we want around the outside of an element.
  • A 'padding' indicates how much space we want around the content inside an element.
  • A 'box-shadow' adds shadow effects around an element's frame.
  • An 'img' selector assigns property values to any image element in the markup.
  • A 'display' property defines how the components such as hyperlink, heading, div, etc are going to be placed on the web page.

Git

  • Enter your Git notes here
    • 'git pull' fetch and merge any commits from the tracking remote branch.
    • 'git status' checks what branch we are currently on.
    • 'git checkout -b branch-name:' creates a new branch and switches to it.
    • 'git clone' clones all the folders, files and .git files that contains Git version control.
    • 'code .' opens the folder in VS Code on the current directory
    • For more information about git commands please visit: https://training.github.com/downloads/github-git-cheat-sheet/

JavaScript

  • Enter your JavaScript notes here
  • A 'variable' is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An 'array' is a single variable that is used to hold a group of data.
    • Arrays are typically used to hold data that is related in some way.
    • Array index is the location of an item in an array.
  • A 'loop' is used to perform repeated tasks based on a condition. Conditions typically return true or false.
    • A 'for loop' uses the predictable pattern of indices to perform a task on all items in an array by allowing a single code block to be executed over and over.
    • A loop will continue running until the defined condition or 'conditional statement' returns false.
    • A 'conditional statement' is used for checking a specific condition and executing the code based on the specified condition.
